# DishCover Frontend

Aplicación móvil con React Native, Expo Router y NativeWind para DishCover.

Desarrollado por Manuel García Nieto.

## Arranque

```bash
npm install
npm start ó npx expo start
```

La primera entrega usa una navegación funcional con pantallas de onboarding, inicio de sesión, registro, recetas, favoritos, eventos y perfil. La integración completa se completará en la PEC4.
